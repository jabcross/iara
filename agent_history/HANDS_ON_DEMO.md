## Part 3: Understanding the Generated Configuration

### Step 3.1: Analyze Parameter Combinations

```bash
# Use Python to understand what parameter combinations were generated
python3 << 'EOF'
from tools.experiment_framework import load_experiments_yaml, get_parameter_combinations
from pathlib import Path

# Load the configuration
config = load_experiments_yaml(Path("applications/05-cholesky/experiment/experiments.yaml"))

# Get combinations for regression set
combos = get_parameter_combinations(config, "regression")

print(f"Total combinations: {len(combos)}\n")

# Show first 5
print("First 5 combinations:")
for i, combo in enumerate(combos[:5], 1):
    print(f"  {i}. {combo}")

# Analyze by scheduler
by_scheduler = {}
for combo in combos:
    sched = combo['scheduler']
    if sched not in by_scheduler:
        by_scheduler[sched] = 0
    by_scheduler[sched] += 1

print("\nBreakdown by scheduler:")
for sched in sorted(by_scheduler.keys()):
    print(f"  {sched}: {by_scheduler[sched]} instances")
EOF
```

**Expected output:**
```
Total combinations: 42

First 5 combinations:
  1. {'scheduler': 'sequential', 'matrix_size': 2048, 'num_blocks': 1, 'block_size': 2048}
  2. {'scheduler': 'sequential', 'matrix_size': 2048, 'num_blocks': 2, 'block_size': 1024}
  3. {'scheduler': 'sequential', 'matrix_size': 2048, 'num_blocks': 4, 'block_size': 512}
  4. {'scheduler': 'omp-for', 'matrix_size': 2048, 'num_blocks': 1, 'block_size': 2048}
  ...

Breakdown by scheduler:
  omp-for: 12 instances
  omp-task: 12 instances
  sequential: 12 instances
  vf-omp: 6 instances
```

### Step 3.2: Understand Instance Naming

```bash
# Show instance naming pattern
python3 << 'EOF'
import re
from pathlib import Path

cmakelists = Path("applications/05-cholesky/experiment/CMakeLists.txt").read_text()
names = re.findall(r'NAME "([^"]+)"', cmakelists)

print("Instance Name Structure:")
print("=" * 80)
print(f"{'Instance Name':<50} {'Scheduler':<15}")
print("=" * 80)

for name in names[:10]:
    # Parse the name
    parts = name.split('_')
    app = '_'.join(parts[:-6])  # Everything except last 6 is app
    scheduler = parts[-5]

    print(f"{name:<50} {scheduler:<15}")

print(f"... ({len(names)} total)")
EOF
```

**Key insight:** The instance name encodes:
- Application: `05-cholesky`
- Scheduler: `sequential`, `omp-for`, `omp-task`, `vf-omp`
- Parameters: sorted alphabetically (matrix_size, num_blocks)

**Output:**
```

```

### Step 6.2: See How Change Detection Works

```bash
python3 << 'EOF'
from tools.experiment_framework import compute_yaml_hash, check_yaml_changed
from pathlib import Path

yaml_path = Path("applications/05-cholesky/experiment/experiments.yaml")
hash_file = Path("applications/05-cholesky/experiment/.experiments.yaml.hash")

# Current hash
current_hash = compute_yaml_hash(yaml_path)
print(f"Current YAML hash: {current_hash}")

# Check if changed
changed = check_yaml_changed(yaml_path, hash_file)
print(f"YAML changed: {changed}")

# Show the stored hash
stored_hash = hash_file.read_text().strip()
print(f"Stored hash:  {stored_hash}")
print(f"Match: {current_hash == stored_hash}")
EOF
```

**Output:**
```
Current YAML hash: 2178138e46af1df4b543c625cdd1b252a20915b1603bcf47e6a4502f87b7cc30
YAML changed: False
Stored hash:  2178138e46af1df4b543c625cdd1b252a20915b1603bcf47e6a4502f87b7cc30
Match: True
```

## Part 8: Python API - Direct Usage

### Step 8.1: Use the Framework API Directly

```python
python3 << 'EOF'
from tools.experiment_framework import (
    load_experiments_yaml,
    get_parameter_combinations,
    generate_cmakelists,
    compute_yaml_hash,
    check_yaml_changed
)
from pathlib import Path

# 1. Load configuration
yaml_path = Path("applications/05-cholesky/experiment/experiments.yaml")
config = load_experiments_yaml(yaml_path)

print("1. Configuration loaded")
print(f"   Application: {config['application']['name']}")
print(f"   Parameters: {len(config['parameters'])}")
print(f"   Experiment sets: {len(config['experiment_sets'])}")

# 2. Get parameter combinations for a specific set
combos = get_parameter_combinations(config, "regression")
print(f"\n2. Parameter combinations for 'regression': {len(combos)}")

# 3. Generate CMakeLists.txt
output_path = Path("applications/05-cholesky/experiment/CMakeLists.txt")
num_instances = generate_cmakelists(yaml_path, output_path, "regression")
print(f"\n3. Generated CMakeLists.txt with {num_instances} instances")

# 4. Check hash
hash_value = compute_yaml_hash(yaml_path)
print(f"\n4. YAML hash: {hash_value[:16]}...")

# 5. Verify hash
hash_file = Path("applications/05-cholesky/experiment/.experiments.yaml.hash")
changed = check_yaml_changed(yaml_path, hash_file)
print(f"   File changed: {changed}")
EOF
```

**This shows how to use the framework programmatically**
